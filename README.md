## 使用

### 启动后端

进入 server 目标，运行如下命令：

```
nodemon index.js
```

使用 nodemon 的目的是让 index.js 热更新。
=======
通过这个项目理清楚 ajax 跨域的知识点。

## 安装

```
npm install
```

其中：

1. express 是方便快速提供 node 服务器功能。可以使用原生的 node 代码来搭建。
2. nodemon 是为了热更新 index.js 代码。

1. 前端

```
npm run front
```

2. 后端

